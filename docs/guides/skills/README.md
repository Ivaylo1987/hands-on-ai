# Building Skills for Claude — Interactive Guide

An interactive, comprehensive guide on how to build custom skills for Claude, based on [Anthropic's official documentation](https://resources.anthropic.com/hubfs/The-Complete-Guide-to-Building-Skill-for-Claude.pdf) and the [latest API docs](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview).

## [View the live guide →](https://ivaylo1987.github.io/ai-coding/guides/skills/)

## What's inside

A 7-chapter journey that takes you from zero to a production-ready skill:

| Chapter | Topic | What you'll learn |
|---------|-------|-------------------|
| 1 | **Understand** | What a skill is, folder structure, progressive disclosure |
| 2 | **Plan** | Use cases, skill categories, success criteria |
| 3 | **Structure** | Naming rules, YAML frontmatter, security restrictions |
| 4 | **Write** | Description field, SKILL.md template, writing principles |
| 5 | **Patterns** | 5 proven patterns with code, MCP + Skills integration |
| 6 | **Test** | Triggering/functional/performance testing, skill-creator, Claude A/B pattern |
| 7 | **Ship** | Upload paths, team sharing, troubleshooting, positioning |

Plus an interactive **skill validator** that checks your SKILL.md against the official checklist.

## Interactive features

- **Folder explorer** — click to learn what each file does
- **Progressive disclosure widget** — see the 3-level loading system in action
- **Pattern accordion** — 5 patterns with full code examples
- **Troubleshooting cards** — click to reveal fixes
- **Skill validator** — paste your SKILL.md and get instant feedback
- **Quick reference panel** — floating journey nav + links
